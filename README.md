## Getting Started

Check your environment:

```yaml
Hexo: 5.4 ~ 6.x
hexo-cli: 4.3 ~ latest
node.js: 16.x LTS ~ latest LTS
npm: 8.x ~ latest LTS
```

Edit your `_config.yml`:

```yaml
theme: volantis
```

Install Volantis in terminal:

```bash
npm i hexo-theme-volantis
```
